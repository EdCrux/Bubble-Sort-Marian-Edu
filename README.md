## Getting Started

**You can download the zip file, o take a look on the live demo**

### Prerequisites

- Ruby version ~> 2.6
- IDE (if you want to test)

### Dependencies

- Rubocop

### Install

`gem rubocop` `bundle install`


### Run tests

Rubocop lint
` rubocop .`

### Deployment

`ruby bubble_sort.rb`
